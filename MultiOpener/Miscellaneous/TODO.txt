========== OVERALL FEATURES ===========
-- GENERAL
   - Custom Dialog Boxes
-- Start
   - Opening presets
     - When opening with pressed shift it will open structure of all 'opens' in preset
   - Cancelling preset opening process
     - After cancelling opening it will open rest of 'opens' as structure
   - Controlling "opens" within an opened preset
     - Viewing information about the process
       - Opening the "open" directory
       - Performing actions based on the type of "open"
     - Resetting the current "open"
     - Closing/opening the current "open"
   - Refreshing all opened preset "opens"
     - Normal refresh: looking for the HWND of an "open" process, just looking if that process is opened in background but not detected for example by MultiOpened etc etc
     - Fast refresh: quickly checking if an "open" process is closed or still open
     - Option to cancel refresh
-- Presets
   - Adding a new program/process to run as an "open"
     - Main options: Type, App path, Delays before and after, minimize on open
     - Additional options depend on the "open" type
   - Rearranging the execution order of "open" items in the list by dragging them
   - Clearing all "open" options
   - Removing an "open" item
   - Creating a new empty preset
   - Deleting a whole preset
   - Opening the folder containing all created presets
   - Loading a preset
   - Saving a preset
     - Overwriting the currently loaded preset with a new name
     - Creating a new preset based on the current loaded preset with a different name
-- Options
   - Resetting all options to default
   - Changing general options
     - (Empty)
   - Changing timings
     - Main timings
       - Late refresh time after all "opens" in a preset have finished opening
       - Opening timeout for each "open" process while opening the preset
       - Opening timeout for a selected "open" process
     - MultiMC instance timings
       - Timeout for looking up instance data to obtain HWND for an MC instance
       - Timeout for finalizing instance data after finding all HWNDs for all instances, allowing for a cooldown period for MCs to open
       - Timeout for opening a single instance
